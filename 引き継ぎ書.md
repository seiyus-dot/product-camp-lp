# PRODUCT AI CAMP 申込フォーム 引き継ぎ書

## 概要
PRODUCT AI CAMP の申込フォームです。
URL: `https://product-camp-lp.vercel.app/apply`

---

## 追加・変更したファイル

| ファイル | 説明 |
|---------|------|
| `components/ApplyForm.tsx` | 申込フォームのUIコンポーネント |
| `app/apply/page.tsx` | 申込フォームのページ（/apply） |
| `app/api/apply/route.ts` | Stripe決済APIルート |
| `package.json` | stripe, @stripe/react-stripe-js, @stripe/stripe-js を追加 |

---

## フォーム入力項目

- 氏名（必須）
- メールアドレス（必須）
- 電話番号（必須）
- 住所（必須・契約書送付先）
- 参加日程（必須・4月11-12日 or 4月25-26日）
- カード名義人（必須）
- クレジットカード情報（必須）
- 利用規約同意（必須）

---

## 決済仕様

- 金額：¥200,000（税込）固定
- 決済方法：Stripe PaymentIntent（即時課金）
- 顧客情報はStripeのCustomerとして保存される

---

## 環境変数（Vercelに設定が必要）

| 変数名 | 説明 |
|--------|------|
| `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` | Stripe公開キー（`pk_live_...`） |
| `STRIPE_SECRET_KEY` | Stripeシークレットキー（`sk_live_...`） |

### Vercelへの設定方法
1. Vercelダッシュボード → `product-camp-lp` プロジェクト
2. 「Settings」→「Environment Variables」
3. 上記2つを追加
4. 再デプロイ

---

## ローカル開発

```bash
npm install
npm run dev
```

`.env.local` にStripeのテストキーを設定してください：
```
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
STRIPE_SECRET_KEY=sk_test_...
```

テストカード：`4242 4242 4242 4242`（有効期限：未来の日付、CVC：任意3桁）

---

## 日程変更の方法

`components/ApplyForm.tsx` の先頭にある `DATES` を編集：

```typescript
const DATES = [
  { id: "apr11-12", label: "4月11日（土）・12日（日）" },
  { id: "apr25-26", label: "4月25日（土）・26日（日）" },
];
```

---

## 金額変更の方法

`app/api/apply/route.ts` の `amount` を変更（単位：円）：

```typescript
amount: 200000,  // ← ここを変更
```
